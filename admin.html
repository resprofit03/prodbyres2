Вход

Страница доступна по адресу /admin (на Vercel) или admin.html локально. В меню сайта ссылки нет — только прямой вход. Пользователь в Supabase должен совпадать с adminEmail в js/config.js.

Выполните в SQL Editor скрипт supabase/rls-main-admin.sql, чтобы только этот email мог менять галерею и загружать файлы.
